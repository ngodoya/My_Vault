# Cómo Realizar cambios en un Repo?

Remitase a la carpeta del Repo → <>Code → Codespaces +

## Comando para la Consola y guardar cambios.

```bash
# Utilice la sintax
git add.
git commit -m'mensaje' ##Dejar REgistro
git push
# Modificar Commits (OJO)
git rebase -i HEAD~3 #Quiero modificar los últimos 3 commits
```
